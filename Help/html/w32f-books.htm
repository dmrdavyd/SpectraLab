Forth tutorials & ebooks

Besides an excerpt from the Win32Forth Help core and description of ANS Forth Standard, this Help system includes three general Forth programming guides that may be instrumental for users interested in employing all the advanced features offered by the SpectraLab scripting language.

  • Perhaps the easiest, most straightforward, and the least time-consuming introduction to Forth is provided in the "And So Forth guide by J. L. Bezemer.
  • Another valuable introduction to Forth programming is  "Programming Forth" book by Stephen Pelc
  • "A beginner's guide to Forth" by J. V. Noble might be less reader-friendly than the above two tutorials. However, this guide contains essential information about Win32Forth, the Forth compiler used in  SpectraLab.
